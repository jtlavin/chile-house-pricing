# ML Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and experimentation.

## Recommended Notebook Order

1. **01_eda.ipynb** - Exploratory Data Analysis
   - Load and explore the scraped property data
   - Understand distributions, correlations, and patterns
   - Identify data quality issues

2. **02_feature_engineering.ipynb** - Feature Engineering
   - Create new features from existing data
   - Calculate price per mÂ², distance features, etc.
   - Test different feature transformations

3. **03_model_experiments.ipynb** - Model Experiments
   - Train baseline models
   - Compare different algorithms (Linear Regression, XGBoost, LightGBM)
   - Hyperparameter tuning
   - Evaluate model performance

## Getting Started

Install Jupyter:
```bash
pip install jupyter
```

Start Jupyter:
```bash
jupyter notebook
```
